üß† About AST-Based Analysis

  • Abstract Syntax Tree (AST): Analyzes code structure rather than surface-level text
  • Variable Normalization: Detects identical logic even with different variable names
  • Structural Comparison: Uses tree edit distance to measure code similarity
  • Thresholds: >75% = High risk, 60-75% = Medium risk, <60% = Low risk
  • Current Quiz: Compares all student submissions within this quiz
  • All Quizzes: Identifies students reusing code across different quizzes
